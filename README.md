# מש"ב – Military Scheduling Frontend Prototype

פרויקט React מוכן לאב טיפוס של מערכת שיבוץ ביטחונית.

## מה יש בפרויקט
- מסך התחברות
- דשבורד
- ניהול זמינות
- שיבוצים פעילים
- יצירת שיבוץ חדש
- דיווח נוכחות
- התראות וזימונים
- דוחות
- ניהול משתמשים
- פרופיל משתמש

## איך להריץ
1. התקיני Node.js
2. פתחי טרמינל בתוך התיקייה
3. הריצי:

```bash
npm install
npm run dev
```

## הסבר קצר על הארכיטקטורה
- `src/pages` – כל מסך במערכת
- `src/components` – רכיבים חוזרים
- `src/data/mockData.js` – נתוני דוגמה
- `src/styles/global.css` – עיצוב כללי

## איך להמשיך מכאן
- לחבר Backend אמיתי
- להחליף Mock Data ב-API
- להוסיף אימות משתמשים
- להוסיף לוגיקת שיבוץ מלאה
- להוסיף גרפים אמיתיים בדוחות
